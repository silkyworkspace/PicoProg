# 画面遷移図

> S03〜S11 は未ログイン時に自動的に S01（ログイン画面）へリダイレクトされます。
> S11（管理者パネル）は管理者権限のないユーザーがアクセスすると 403 エラー（S12）になります。

```mermaid
flowchart TD
    START([アプリ起動 / 未ログイン])

    S01[S01\nログイン画面]
    S02[S02\n新規登録画面]
    S03[S03\nタイムライン画面]
    S04[S04\n新規投稿画面]
    S05[S05\n投稿編集画面]
    S06[S06\nコメント画面]
    S07[S07\nお気に入り一覧画面]
    S08[S08\n通知画面]
    S09[S09\nプロフィール画面]
    S10[S10\nユーザー設定画面]
    S11[S11\n管理者パネル画面]
    S12[S12\nエラー画面\n404 / 500]

    %% 起動・認証
    START -->|未ログイン| S01
    S01 -->|ログイン成功| S03
    S01 -->|新規登録リンク| S02
    S02 -->|登録成功| S03
    S02 -->|ログインリンク| S01

    %% タイムラインからの遷移
    S03 -->|投稿するボタン| S04
    S03 -->|編集ボタン（自分の投稿）| S05
    S03 -->|コメントボタン| S06
    S03 -->|お気に入りページへ| S07
    S03 -->|通知ページへ| S08
    S03 -->|ユーザー名クリック| S09
    S03 -->|設定| S10
    S03 -->|管理者パネル（管理者のみ）| S11
    S03 -->|ログアウト| S01

    %% 投稿・編集
    S04 -->|投稿完了 / キャンセル| S03
    S05 -->|更新完了 / キャンセル| S03

    %% コメント
    S06 -->|コメント投稿| S06
    S06 -->|トップに戻る| S03

    %% お気に入り一覧
    S07 -->|コメントボタン| S06
    S07 -->|編集ボタン（自分の投稿）| S05
    S07 -->|トップに戻る| S03

    %% 通知
    S08 -->|通知カードクリック| S06
    S08 -->|トップに戻る| S03

    %% プロフィール
    S09 -->|トップに戻る| S03

    %% 設定
    S10 -->|プロフィール・パスワード更新| S10
    S10 -->|トップに戻る| S03

    %% 管理者パネル
    S11 -->|ユーザー・投稿削除| S11
    S11 -->|トップに戻る| S03

    %% エラー
    S12 -->|トップへ戻る| S03
```
